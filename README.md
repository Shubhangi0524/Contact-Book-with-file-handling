A Python-based Contact Book application that demonstrates file handling operations for storing, searching, updating, and deleting contact records.

# 📒 Contact Book using File Handling

A simple Contact Book application developed in Python to practice **file handling** concepts. The application stores contact information in a text file, allowing users to manage contacts without using a database.

---

## 📖 Overview

This project demonstrates Python file operations to create a basic contact management system. Contact details are stored in a text file, making the data persistent even after the program exits.

The project was developed as part of my Python learning journey to strengthen my understanding of file handling and basic CRUD operations.

---

## ✨ Features

- ➕ Add a New Contact
- 📋 View All Contacts
- 🔍 Search for a Contact
- ✏️ Update Contact Details
- ❌ Delete a Contact
- 💾 Store Contacts in a Text File
